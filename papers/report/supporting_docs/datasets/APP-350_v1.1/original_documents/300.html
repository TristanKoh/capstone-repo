App Behaviors

Explanation of required permissions by the APK

  1. android.permission.INTERNET
    android.permission.ACCESS_NETWORK_STATE
    android.permission.ACCESS_WIFI_STATE

    These 3 permission is required by Perfect Piano to access internet. The use cases include:
    1. Download MIDI file for Learn to Play Mode
    2. Onlne Multi-Player Mode to play piano online with other user
    3. Contest Mode to get Achevement and Ranking list
    4. Google Admob Service to get AD Banner content

  2. android.permission.RECORD_AUDIO

    This permission allow Perfect Piano to record through phone MIC. The use case is when you playing the piano keybaord even sing along, You can record the sound at same time.

  3. android.permission.READ_EXTERNAL_STORAGE
    android.permission.WRITE_EXTERNAL_STORAGE

    These 2 permission is required by Perfect Piano to access external storage(or SD card). The use cases include:
    1. Save downloaded MIDI file to external storage
    2. Save MIDI recordings to external storage
    3. Save Audio recordings to external stroage
    4. Read and Playback recordings from external stroage
    5. Load instrument plugins' sound from external stroage

  4. android.permission.WRITE_SETTINGS
    This permission allow Perfect Piano to set midi/audio recordings as Ringtone.

  5. android.permission.VIBRATE
    This permission allow Perfect Piano to vibrate when press keys on piano keyboard. This will make piano keyboard more responsive.

  6. com.android.vending.BILLING
    This permission is required by Perfect Piano to remove advertisement banner by pay 0.99 USD through Google Play.
